My Portfolio-Building Journey

October 11, 2025

I will be talking about my journey, utilizing CSS, HTML, and Javascript to design my first personal website.

Getting Started

When I first decided to create my personal website, I had no idea where to begin. I knew I wanted to showcase my projects and skills, but the technical aspects seemed overwhelming at first.

Learning the Basics

I started by learning HTML to structure my content, then moved on to CSS for styling. The most challenging part was understanding how to make the website responsive and look good on different devices.

Adding Interactivity

Once I had the basic structure and styling down, I began learning JavaScript to add interactive elements. This is where I discovered the power of DOM manipulation and how it can bring a static website to life.

Challenges and Solutions

One of the biggest challenges was understanding how different browsers interpret CSS and JavaScript. I spent countless hours debugging layout issues and ensuring cross-browser compatibility.

What I Learned

This project taught me the importance of planning before coding, the value of testing across different devices, and how rewarding it can be to see your ideas come to life on the web.
